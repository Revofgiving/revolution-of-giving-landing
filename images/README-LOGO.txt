🎨 COME SOSTITUIRE IL LOGO

Per utilizzare il tuo logo originale 'Revolution of Giving':

1. Salva la tua immagine del logo in questa cartella (images/)
2. Rinomina il file in: revolution-of-giving-logo.svg
   (oppure revolution-of-giving-logo.png se preferisci PNG)
3. Il file dovrebbe avere:
   - Sfondo trasparente
   - Dimensioni quadrate (es: 200x200px o 400x400px)
   - Formato SVG (raccomandato) o PNG

📝 NOTA: Attualmente è in uso un logo placeholder SVG creato per simulare
il design del tuo logo originale. Sostituiscilo con la tua immagine reale
per ottenere il risultato finale perfetto!

🎯 Il logo apparirà in:
- Header della pagina (piccolo)
- Sezione hero centrale (grande)
- Footer (medio)
- Loading screen
